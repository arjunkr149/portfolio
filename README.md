# 🌐 Arjun Kumar — Developer Portfolio

- 🌗 Dark / Light theme toggle
- ⌨️ Terminal easter egg — press `Ctrl+K`
- 🔽 Project filter (Hackathon / Academic / Personal / Security)
- 📱 Fully responsive — Bootstrap 5 grid
- 🎞️ AOS scroll animations
- 📬 Formspree contact form
- 🤖 Typing animation
